HTML Outline

HTML Documents are described by HTML Tags
To write in bold, use the "strong" tag.
Use "em" (emphasize) tag for Italics / emphasize
use "u" tag for underlined text.

Tags mark up the beginning and end of an element, using the general syntax: "tagname" content "/tagname"
Where the tagnames are enclosed in less-than and greater-than brackets and not the quotes used for example.

Basic HTML Structure

The first thing in the HTML file should be "!DOCTYPE html" which tells the browser the document's type is html5.
then "html" tag
Indent heading and use the"head" tag, which contains the tags and elements: title, scripts, (css) styles, and meta info - search engine info.
Keeping same indentation, use the "body" tag for anything you want to be directly visible on the web page.
Indent again (2 indents) for "h1" "h2" ... "h6"(h6 is smallest) to write a Header Tag.
Use Paragraph "p" tag for text body like this paragraph.

HTML Syntax above


This is a "h6 align="right" " header

Attributes are just additional info about an element.

To insert an image use the "img" tag with attribute 'src'




To insert a hyperlink use "a" tag with "href="

For example: "a href="http://www.SiteName.com">Hyperlink display name" "/a"

To have link open in a new tab use targets="_blank" after the href= link. Which can be used for a website or even an image.
For example:

"a href="logo.jpg" target="_blank">HTML Syntax "/a"

Cascading Style Sheets (CSS)

CSS defines exactly how HTML looks and allows to make a clear separation between content (HTML) and style (CSS)

Three ways to use CSS

1. CSS code written insdie an HTML tag using 'style='. (i.e., p style="font_size: 120%")

2. CSS code written in the head section of HTML doc by using the "style" tag.
(i.e., "style"
(indent) p {
(indent)font-size: 120%
(dedent)}
(dedent) "/style")

3. Most Practical - CSS can be put into an external file. (i.e., open new file and save it as "style.css")

Linking style.css to your index.html file

In the "head" tag section of your html file, use the "link" tag, which doesn't have an ending tag ("/link" is not used)

For Example:

link rel="stylesheet" type="text/css" href="style.css"

Coding inside the CSS file itself

CSS itself is written in rules, which consists of a selector and a declaration block

Start by formatting the main heading, using the selector "h1" which will select all "h1" elements and format them.
For Example

h1, h2 { This is for things h1 and h2 have in common
color: green;
font-family: Helvetica Neue, Arial;
}

h1 {
font-size: 40px;
}

h2 {
font-size: 25px;
}

Quick Edit a CSS file while inside your HTML file using ctrl+E on the tag you want to format.

The RGB Model

In the RGB Model, each color is defined using a combination of Red, Green, and Blue. Hexadecimal notation is used #RRGGBB

For each color:
Min: 0 (hex 0)
Max: 255 (hex ff)

Some Hexadecimal color codes



If all 6 numbers in hexadecimal code are the same, instead of typing #555555 we can write #555

Transparency with RGBA

Ex: rgba(29, 167, 23, 0.75)

This shows a green rectangle with 75% transparency

You can adjust each color in the CSS file by clicking the color name or hex and using the hot key ctrl+E which opens up a quick edit for that color.
Here you're able to adjust the transparency and color shade.
